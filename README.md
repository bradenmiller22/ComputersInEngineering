## ğŸ§ª How to Build and Run

1. **Clone the repository**:
   ```bash
   git clone https://github.com/bradenmiller22/ComputersInEngineering.git

   ```

2. **Build the project using CMake**:
   ```bash

   cd build
   cmake ..
   ```

